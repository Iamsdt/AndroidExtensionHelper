## How to Config

Add below code in your project level **build.gradle** file

```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

Add this line in your app level **build.gradle** file

```css
implementation 'com.github.Iamsdt:AndroidExtensionHelper:v2.1'
```
